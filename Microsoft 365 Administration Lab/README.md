Overview

In this phase of my IT Support Home Lab, I practised day-to-day Microsoft 365 administration tasks using Microsoft 365 Admin Center, Exchange Online and Microsoft Entra ID.

The lab was built around realistic Help Desk scenarios including user onboarding, password resets, mailbox management, identity administration and access troubleshooting.

Technologies:

    Microsoft 365 Admin Center
    Microsoft Entra ID
    Exchange Online
    Microsoft 365 Business Basic
    Microsoft Entra ID Free

Skills Covered:

    Microsoft 365 User Administration
    Exchange Online Administration
    Microsoft Entra ID
    Security Groups
    Identity Management
    User Lifecycle Management
    Help Desk Troubleshooting
    Access Review
    Cloud Administration
