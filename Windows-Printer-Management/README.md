Overview

In this lab, I practised common printer troubleshooting tasks in Windows 11. These are common issues handled by IT Support and Help Desk technicians when users are unable to print documents.

Objectives:

        Add a printer
        Remove a printer
        Set a default printer
        Restart the Print Spooler service
        Verify printer status
        Troubleshoot printing issues

Environment:

        | Device                 | Role               |
        | ---------------------- | ------------------ |
        | Windows11-PC           | Client Workstation |
        | Microsoft Print to PDF | Test Printer       |
