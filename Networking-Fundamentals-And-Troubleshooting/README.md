Key Interview Notes

        - Port 80 = HTTP
        - Port 443 = HTTPS
        - Port 445 = SMB
        - Port 3389 = RDP
        - LISTENING = waiting for connections
        - ESTABLISHED = active connection
        - TIME_WAIT = recently closed connection
